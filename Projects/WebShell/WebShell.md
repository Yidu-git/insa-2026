# Project summary
This project is meant to help exploit [[Web Shell]] vulnerabilities. It provides a simple terminal interface to exploit web shells by emulating an SSH connection (*without live terminal feedback*).

# Plan
## Exploitation framework:

1. User provides target IP, service port, and exploit URL
2. Script scans for input fields and common reflected template vulnerabilities
3. Script tests for a terminal response
4. Script Provides shell
5. Script keeps track of directory and combines commands to simulate terminal
6. Script logs findings automatically as a markdown file and *CSV*
## Options

- Logging
- Common vuln scans
- Tool integration
- SQLi attacks
