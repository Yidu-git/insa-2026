# Definitions
- **Virtualization** : Emulating or running an Operating system on top of an existing OS, while partitioning memory and storage to a virtual machine/computer.
- **CLI** (Command Line Interface) : is a form of UI (User interface) that uses basic text and possibly colors to provide an interface for the user to interact with
# Linux
## Why Linux
- Open source
- Provides control
- Built in security tools
- Ability to chain programs
- Collaboration
- Customization
## Linux file system
- Has one root hierarchy
- ==hierarchical tree structure== 
- Different directories serve different purposes
#### Directories
- **bin** command binaries
- **boot** boot loader
- **dev** device files (drivers)
- **etc** Host specific system wide config files
- **home** User home
- **lib** Shared library
- **media** Media files (CD-ROM)
- **mnt** 
- **tmp** temporary files
- **usr** Unix system resources
- **var** logs and variable data
- **opt** Add on application packages
- **proc**
- **sys** virtual systems to provide info on system processes
## File commands
- **cat** (prints and duplicates files)
- **echo** (prints and writes to files)
- **touch** (creating an empty file if file does not exist)
- **cp** (duplicate file)
- **mv** (moves and renames files)
- **rm** (removes file) (sudo rm -rf)
- **history**
## Common commands
- **`ls -la`** lists all files in working directory (`pwd`) 
- `chmod` changes permission
	- `chmod 700 file.txt` : private
	- `chmod 777 file.txt` : public
- `chgrp` changes group of file
