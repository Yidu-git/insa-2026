**Web proxy** : a service that intercepts and forwards requests and responses.

# HTTP methods
- **GET** - requests a resource
- **POST** - uploads or sets a resource
- **DELETE** - deletes a resource
- **PUT** - updates an entire resource
- **OPTIONS** - returns the options or permissions on a resource 
- **TRACE** - Performs a loop-back test to see how a request is modified by proxies.
- **PATCH** - partially updates a resource

# HTTP Status codes
- 100 - Informational
	- 100 Continue
- 200 - success
	- 200 resource Successfully retrieved
- 300 - redirect
- 400 - user side issue
	- 401
	- 403 forbidden
	- 404 resource not found
- 500 - server side issue
	- 500 server error
	- 502 Bad gateway
	- 503 Service unavailable
